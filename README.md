A web-based app for learning Linux commands and distros 

## Requirements
- [Node.js](https://nodejs.org) — download and install it first (free, one-time setup)

**Run the app**
- **Windows:** double-click `launch.bat`
- **Linux / Mac:** double-click `launcher.sh`

> Dependencies install automatically on first run — no extra steps needed!

## Features
- Browse common Linux commands
- Learn about popular Linux distros
- Clean and simple interface
